# 🌿 GreenNest – Plant Store E-Commerce Website

A simple and responsive plant store web application built using **HTML, CSS, and JavaScript**.  
It allows users to browse plants, add items to cart, manage purchases, and experience a smooth shopping flow.

A beginner-friendly frontend project focused on **UI design, shopping cart logic, and local storage functionality**.

---

## ✨ Features

- 🌱 Plant product listing page  
- 🛒 Add to Cart functionality  
- ❌ Remove items from cart  
- 🔢 Dynamic cart count update  
- 💰 Total price calculation  
- 💾 Local storage support  
- 📱 Fully responsive design  

---

## ⚙️ Core Functionality

- Products displayed in structured UI cards  
- Users can add/remove items dynamically  
- Cart automatically updates quantity and total price  
- Data stored using browser localStorage  
- Cart persists even after page refresh  

---

## 👤 User Experience

- Clean and modern interface  
- Smooth and fast interactions  
- Easy navigation between pages  
- Mobile-friendly responsive layout  
- Simple and intuitive shopping flow
